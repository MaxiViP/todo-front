# Todo App – Современный менеджер задач

Чистый, быстрый и красивый фронтенд на **Nuxt 4** для управления задачами.

## Стек технологий

- **Nuxt 4** (SSR + Nitro)
- **Pinia** — состояние
- **Nuxt UI** + **Tailwind CSS v4**
- **Axios** + автоматический refresh токена
- **Lucide** иконки
- **TypeScript**
- **VueUse** (debounce)

## Основные возможности

- Полноценная авторизация (логин + remember me)
- Автоматический refresh JWT-токена (через cookie)
- QR-логин (`loginWithToken`)
- CRUD задач с пагинацией (12 на страницу)
- Поиск с debounce 350 мс
- Фильтр по статусу и сортировка
- Защищённые маршруты (middleware)
- Красивый UI на Nuxt UI + кастомные CSS-переменные
- Оптимизированный production-бандл (manual chunks, gzip, кэширование)

## Установка и запуск

```bash
# 1. Клонируем
git clone <repo>
cd todo-frontend

# 2. Устанавливаем зависимости
npm install

# 3. Запускаем разработку
npm run dev

# 4. Сборка
npm run build

## Для удобства тестирования реализован быстрый вход
При нажатие на qr  можно выбрать тестового пользователя
```
